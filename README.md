## ğŸ§  System Architecture (RAG Pipeline)

The application follows a standard Retrieval-Augmented Generation flow:

1. **PDF Ingestion**  
   Users upload one or more PDF documents via the Streamlit UI.

2. **Chunking**  
   Documents are split into overlapping text chunks using recursive character splitting.

3. **Embedding**  
   Text chunks are embedded using a Hugging Face sentence-transformer model.

4. **Vector Search (FAISS)**  
   Relevant chunks are retrieved via FAISS similarity search (no database persistence).

5. **Answer Generation**  
   Claude generates answers **only from retrieved context**, preventing hallucination.

6. **Source Display**  
   The app shows source document names, page numbers, and excerpts alongside answers.
